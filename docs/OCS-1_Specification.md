# OCS-1: Core Specification  
## 1. Color Notation  
Format: `OCS-[ID]-[C]-[M]-[Y]-[K]-[O]-[G]`  
- **Rules**:  
  - Omit trailing zeros: `0-800-200` instead of `0-800-200-0-0-0`.  
  - Use `FULL` for 1000g: `OCS-RED-FULL-0-0`.  

## 2. Base Inks  
| Ink | CIELAB Reference |  
|-----|------------------|  
| Cyan | `L=50, a=-30, b=-50` |  
| Magenta | `L=50, a=70, b=-40` |
